표 만들기

 
	[태그]

	table : 표의 시작, 표 전체
	tr : 1줄 생성
	td : 1칸 생성
	th : 강조하고 싶은 줄에 td대신에 th 사용(글자가 칸의 가운데 정렬됨, 글자 진하게 )
	
	[열(column) 단위로 처리]
	
	colgroup
	col : 각 칸 단위 , 각 1열
	span : 같은 스타일 적용 "어디 칸까지"
	
	[표 구성]
	
	caption: 표의 제목
	thead : 표의 맨 상단
	tbody : 표의 내용
	tfoot : 표의 맨 하단 
	
	[속성]
	
	width, height : px, %
