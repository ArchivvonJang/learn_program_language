스타일 시트

  • 스타일 표시 방법
  • inline 방식 : html 태그에 style 속성으로 표기
  • internal 방식 : head 태그에 style 태그를 이용하여 표기하는 방법 / 주로 head 태그 안에 모아놓는다.
  • external 방식 : 스타일 시크르 별도의 외부파일로 생성하여 사용한다.
며칠 한겨울 날씨가 이어졌는데요, 내일부터는 차츰 봄에 다시 가까워지겠습니다. 내일 출근길 서울이 영하 5도로 쌀쌀하겠지만 낮부터는 기온이 빠른 속도로 오르겠고요, 서울이 10도로 평년 수준을 크게 웃돌겠습니다. 추위가 풀리고 다가오는 주말과 휴일에도 내내 기온은 오름세를 보이겠습니다. 토요일 서울의 낮 기온이 13도, 일요일에는 15도가 예상되고요, 전국이 봄처럼 포근하겠습니다. 내일 전국적으로 하늘은 맑겠습니다. 다만 밤부터는 중국발 스모그가 유입되겠고요, 중서부지역 중심으로 공기가 탁해지겠습니다. 대기의 건조함까지 심해지고 있는데요, 일부 수도권과 동쪽 지방 곳곳에 건조특보가 발효 중입니다. 화재 사고에 유의하셔야겠습니다.
